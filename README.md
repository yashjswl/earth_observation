# Earth Observation
> **Pipeline for AI-based audit of the Delhi Airshed to identify land use patterns and pollution sources, involving spatial gridding and land cover classification.**

Submission for SRIP-2026 [AI Sustainability] - Yashasvi Jaiswal

---

## Details
- Platform: Jupyter Notebook
- Dataset: Delhi Airshed Sentinel-2 RGB and Landcover [[Link]](https://www.kaggle.com/datasets/rishabhsnip/earth-observation-delhi-airshed)
- Libraries need to be imported
   -  Q1: matplotlib, numpy, pandas, geopandas, math, shapely, os 
   -  Q2: rasterio, scikit-learn, geemap, ee
   -  Q3: torch, PIL
